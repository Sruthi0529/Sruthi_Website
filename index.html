Projects

Lane detection and Vehicle tracking

This algorithm integrates camera calibration, distortion correction, perspective transformation, thresholding, lane pixel detection for lane detection and employs the YOLOv3 algorithm for real-time object detection and utilizes bounding box association techniques to track vehicles across consecutive frames. It applies Non-Maximum Suppression and confidence metrics to enhance the accuracy and reliability of vehicle tracking.

Face detection

This algorithm utilizes the OpenCV library for face detection, converts the image to grayscale, uses a pre-trained Haar Cascade classifier for face detection, and returns a list of bounding boxes representing the detected faces.

Panorama Image

This image stitching algorithm utilizes SIFT for keypoint detection and RANSAC for homography calculation. Iteratively applying homography and warping, it stitches images with a 20% overlapping criterion. The warping process, employing perspective transformations, aligns and combines features from input images.

Classification

Generated synthetic datasets with optional outliers for binary and multi-class classifications, utilizing classifiers like Least Squares, Logistic Regression, and SVM. Applied these methods to real-world datasets such as "Abalone," incorporating SVM with polynomial and RBF kernels, along with variable regularization. The code further employs linear and polynomial kernel SVMs with PCA for dimensionality reduction.

Regression

Generated toy datasets with varying underlying functions and noise levels. It then performs polynomial regression using both linear regression and ridge regression with different polynomial degrees and regularization strengths. The resulting plots showcase the impact of model complexity (polynomial degree) and regularization on the regression fit.
